# RDM Digital Hub — Nodo Cero

**Plataforma digital soberana para Real del Monte, Hidalgo, México**

Sistema de Inteligencia Territorial con arquitectura heptafederada YUN, IA consciente (Isabella Villaseñor AI), criptografía post-cuántica, gemelo digital 2D/3D, gamificación y economía local phygital.

## Inicio rápido

```bash
npm install --legacy-peer-deps
npm run dev
```
