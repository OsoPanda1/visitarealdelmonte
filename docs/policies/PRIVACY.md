# Política de Privacidad — RDM Digital LTOS

## 1. Quiénes somos

RDM Digital LTOS es el Sistema Operativo Urbano de Real del Monte, diseñado por TAMV Online para impulsar el turismo, el comercio local y la organización comunitaria.

## 2. Principios

- **No explotación de datos personales**: no vendemos ni cedemos datos personales a terceros con fines comerciales.
- **Datos primero agregados**: usamos estadísticas agregadas y anonimizadas para mejorar servicios, seguridad y planificación territorial.
- **Soberanía digital**: los datos del territorio pertenecen al territorio; priorizamos infraestructura y decisiones locales.

## 3. Datos que podemos recoger

Dependiendo de cómo uses RDM Digital, podemos procesar datos de uso anónimos, geolocalización aproximada, datos de sesión y datos de cuenta cuando exista registro voluntario (nombre, correo electrónico y rol).

No solicitamos datos sensibles como salud, religión, orientación política, biometría, contraseñas, documentos oficiales o información financiera. Su almacenamiento queda prohibido sin consentimiento explícito, finalidad documentada y controles reforzados.

## 4. Cómo usamos los datos

Usamos los datos para mostrar información relevante, mejorar servicios, generar métricas **agregadas**, detectar incidentes y fortalecer la resiliencia territorial. Siempre que generemos estadísticas, agregamos y/o anonimizamos los datos para evitar atribución a una persona específica.

## 5. Qué no hacemos

- No vendemos tus datos personales.
- No creamos perfiles comerciales individualizados para publicidad externa.
- No compartimos información identificable con terceros, salvo obligación legal.

## 6. Uso de IA

Los asistentes Isabella/Realito AI se alimentan principalmente con información cultural y territorial de Real del Monte. Pueden registrar consultas de forma **anonimizada** para mejorar respuestas y detectar abusos. No deben utilizarse para introducir datos personales sensibles.

## 7. Conservación y seguridad

Los datos se almacenan en proveedores cloud con cifrado en tránsito, controles de acceso y actualizaciones de dependencias. Conservamos los datos sólo durante el tiempo necesario para cumplir la finalidad y obligaciones legales.

## 8. Tus derechos

Según la legislación aplicable, incluyendo la Ley Federal de Protección de Datos Personales en México, puedes solicitar acceso, rectificación, supresión, limitación u oposición. Contacto: `privacidad@rdm.digital`.

## 9. Cambios en esta política

Podemos actualizar esta política para reflejar cambios en el proyecto. Mantendremos historial de cambios en el repositorio.
